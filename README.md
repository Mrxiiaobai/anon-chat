# anon-chat
这是一个聊天室项目

# 运行方式
  单独启动react项目，不启动桌面app
  ```javascript
  npm run dev
  ```

  打包react项目
  ```javascript
  npm run build
  ```

  启动开发环境桌面app
  ```javascript
  npm run electron-dev
  ```

  启动生产环境桌面app
  ```javascript
  npm run electron-dev
  ```

  打包桌面app
  ```javascript
  npm run make
  ```
